# Vertex Overview

> Source: <https://www.palantir.com/docs/foundry/vertex/overview/>
> Captured during the ontology-parity effort. Concrete feature taxonomy only.

## Core visualization

- **Graph exploration** — live network visualization of cause/effect across the digital twin; object-relationship exploration; node/edge display customization.
- **Graph management** — reusable graph templates; persistence (save/share explorations); generation from other Foundry applications.

## Advanced analysis

- **Scenario modeling** — scenario configuration workspace; model chaining (sunset); comparative "what-if" analysis.
- **Time-series integration** — event configuration/visualization; timeline filtering; time-series correlation; threshold/alert config; event-to-object mapping.
- **Media & annotation** — image-layer overlays; annotation/markup tools; media integration in graph context.

## Integration patterns

- **Data connection** — function-backed property derivation; Workshop module embedding; read-only exploration; action invocation from graph context.
- **External integration** — Marketplace packaging; collaborative sharing.
