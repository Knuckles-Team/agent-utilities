# Object Explorer Overview

> Source: <https://www.palantir.com/docs/foundry/object-explorer/overview/>
> Captured during the ontology-parity effort. Concrete feature taxonomy only.

## Search & discovery

- **Text search** — keyword search + comprehensive property filters (dedicated search-syntax + text-search docs).
- **Property filtering** — property-based constraints to locate matching objects.

## Analysis & visualization

- **Charting** — preset and configurable visualizations within the exploration view.
- **SQL-based analysis** — programmatic querying of result sets.
- **Tabular display** — standard data-grid presentation of object sets.

## Object-set operations

- **Comparison** — compare and contrast object sets.
- **Bulk actions** — apply actions (e.g. writeback) on a whole object set.
- **Link traversal (pivot)** — pivot to explore linked objects, navigating relationships within result sets.

## Data management

- **Saved explorations & lists** — save explorations / object lists and revisit; both retain latest results on re-access.
- **Export & integration** — open results in other compatible applications (e.g. Quiver); export for external use.
