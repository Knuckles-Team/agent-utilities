# DSTDD Design Artifacts

This folder contains architectural design artifacts (C4 Context, Mermaid diagrams, pillar interconnection graphs) generated during the **Design phase** of the DSTDD pipeline.

## The Extend-Before-Invent Mandate
All new features, external API integrations, or research paper implementations must first be expressed as an extension of an existing Pillar:
- **ORCH-1.0**: Graph Orchestration
- **KG-2.0**: Knowledge Graph
- **AHE-3.0**: Agentic Harness Engine
- **ECO-4.0**: Ecosystem & Capabilities
- **OS-5.0**: Agent OS Infrastructure

Design diagrams should visualize exactly how a new concept weaves into this existing 5-pillar structure.
